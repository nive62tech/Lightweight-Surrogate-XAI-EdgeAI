# Lightweight-Surrogate-XAI-EdgeAI
Energy efficient explainable AI for Edge devices: A lightweight surrogate model approach



Deployment Instructions (Raspberry Pi)

1. Update system:
   sudo apt update
   sudo apt upgrade -y

2. Install pip:
   sudo apt install python3-pip -y

3. Clone repository:
   git clone https://github.com/YOUR_USERNAME/Lightweight-Surrogate-XAI-EdgeAI.git
   cd Lightweight-Surrogate-XAI-EdgeAI

4. Install dependencies:
   pip3 install numpy
   pip3 install tflite-runtime

5. Run model:
   python3 test_model.py
